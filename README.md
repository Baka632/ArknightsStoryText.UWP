# Arknights Story Text - UWP
一个基于 UWP 的明日方舟纯文本剧情阅读器

## 最低支持平台
Windows 10 1607 (14393)

## 构建需求
- Visual Studio 2017 及以上
- Windows 10 SDK (至少为14393)
    - 需安装 Windows SDK for UWP Managed Apps
